Summary: Latitude vs. Maximum Temperature



The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled datafrom teh OpenWeatherMap API to asssemble a dataset of 500 cities.

After assembling the dataset, we used Matplotlib to put various aspects of the weather vs. lattitude. Factors we looked at included temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any tends and correlations witnessed.
